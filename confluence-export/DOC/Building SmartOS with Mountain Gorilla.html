Setup

Joyent run their Jenkins slaves using the root user, so that's what I'll do here.

Create zone as normal (See Building SmartOS on SmartOS), but use a 30GB quota just to be safe on disk...

Set up SSH keys for at least github and ideally Manta too (or access it over SSH with ssh-agent forwarding).

Get Mountain Gorilla Ready:

pkgin -y in scmgit gmake gcc47
git clone git@github.com:joyent/mountain-gorilla.git MG
cp MG/tools/jenkins-build ./

Building SmartOS

To build the latest master branches of SmartOS:

export JOYENT_BUILD="false"
export JOB_NAME="smartos"
./jenkins-build

Building a release branch

To rebuild a particular release:

export JOYENT_BUILD="false"
export JOB_NAME="smartos"
BRANCH=release-20141211 ./jenkins-build

Mixing branches

Read the jenkins-build script (See https://github.com/joyent/mountain-gorilla/blob/master/tools/jenkins-build#L42-53) for full instructions on how to specify which branch(es) to build.
