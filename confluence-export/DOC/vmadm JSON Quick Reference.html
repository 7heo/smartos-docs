Disks

Property Description Applies Create Update Default
block_size ZVol Block Size KVM Yes No 8192
boot Specifies a bootable disk (Boolean) KVM Yes Yes false
nocreate Don't create disk KVM Yes No false
image_name Name of dataset from this to clone this disk Yes Yes no
image_size Size (MB) of Image being cloned Yes Yes  
image_uuid UUID of dataset to clone Yes Yes  
size Size (MB) of disk to create KVM Yes Yes  
media "disk" or "cdrom" KVM Yes Yes disk
model Disk model (virtio, ide, or scsi) KVM Yes Yes (see disk_driver)
compression Specify compression algorithm for disk ("on,off,lzjb,gzip,gzip-N,zle") KVM Yes Yes off
zpool Zpool on which to create this zvol KVM Yes Yes zones 
"disks": [
        {
          "boot": true,
          "model": "virtio",
          "image_uuid": "e173ecd7-4809-4429-af12-5d11bcc29fd8",
          "image_name": "ubuntu-10.04.2.7",
          "image_size": 5120
        }
      ]

NICs

Property Description Applies Create Update Default
interface Name of the interface within the VM ("net0") Both Yes No
ip IPv4 Address for NIC, or "dhcp". Deprecated, see "ips". Both Yes Yes
ips Array of IPv4 & IPv6 addresses (in CIDR format) for the NIC, "dhcp", or "addrconf". Both Yes Yes
mac MAC Address for virtual NIC Both Yes No Auto-generated
model Driver for this NIC (virtio, e1000, rtl8139) KVM Yes Yes Value of 'nic_driver' property
netmask Netmask for NIC Both Yes Yes  
gateway Gateway for NIC. Deprecated, see "gateways". Both Yes Yes  
gateways Array of gateways for NIC. Both Yes Yes  
nic_tag nic_tag or interface name that the virtual NIC is attached to Both Yes Yes  
primary Sets this NICs gateway as the default gw Both Yes Yes
vlan_id VLAN ID for this NIC Both Yes Yes "0" (None)
vrrp_primary_ip
vrrp_vrid
dhcp_server Allow the VM on this NIC to act as a DHCP Server Both Yes Yes false
allow_dhcp_spoofing Allow this VM to spoof DHCP packets on this NIC Both Yes Yes false
allow_ip_spoofing Allow this VM to spoof IP addresses on this NIC Both Yes Yes false
allow_mac_spoofing Allow this VM to spoof its MAC address Both Yes Yes false
allow_restricted_traffic   Both Yes Yes false
allow_unfiltered_promisc   Both Yes Yes false
blocked_outgoing_ports

Example:

"nics": [
        {
          "nic_tag": "external",
          "ip": "10.2.121.70",
          "netmask": "255.255.0.0",
          "gateway": "10.2.121.1",
          "primary": true
        }
      ]
